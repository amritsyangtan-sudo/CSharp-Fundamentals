# What is Class ?
    A class is a blueprint that groups together the data and behaviour of a single concept.
    Fundamentally class is a user-defined type just like and int, string, double, char, bool etc.

### Analogy
```
    +----------------------+
    |      Blueprint       |
    |----------------------|
    | 3 Bedrooms           |
    | 1 Kitchen            |
    | 2 Bathrooms          |
    +----------------------+

Architect creats a room house blueprint.
We cannot live inside this blueprint but multiple houses can be build form the blueprint.

Class is like that blueprint. It defines the structure. It is not the actual thing.
```

### First Class

```csharp
    class customer
    {
        public string name;
        public string Email;
        public string Phone;
    }

    This structure tells us that ever customer has a name, email and phone.
    Instead of creating a seperate variable for every customer like customer_name1, customer_name2
    customer_name3, customer_name4 , we have create a type that describes what a customer is.

    When compiles sees the class, it records the metadata describing the type.

    At this point.
    - No customer exists.
    - No memory for Name, Email and Phone exists.
```

## Learning
    Programming language has built in types like int, string, char, bool, double etc.
    Now we need a Customer type. language doesnot has the Customer type datatype.
    Therefore, Fundamentally Class is a user-defined type. Compiler now understands Customer type.
    Just like int age is int type,  Customer customer is customer type.
