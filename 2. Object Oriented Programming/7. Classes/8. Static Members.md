# Why `static` ?
The language needed a way to distinguish between :
    belong to one object
    belong to the class itself

Two kinds of members
- Instance Member (Belongs to one object)
```csharp
    class Customer
    {
        public string Name{get;set;}
    }

    Customer customer1 = new Customer();
    customer1.Name = "amrit";

    Customer customer2 = new Customer();
    customer2.Name = "ram";

    Two objects has to different names
```

- Static Member (Belongs to class)
```csharp
    class Customer
    {
        public static int CustomerCount;
    }

    Every created object sees the same value for CustomerCount;
```

# Why static method cannot access instance members but Instance method access static members.
