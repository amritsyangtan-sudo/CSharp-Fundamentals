# Object Creation
```csharp
class Customer
{
    public string Name;
    public string Email;
}

Customer customer = new Customer();

lets break it down:

Customer =  it is a type exactly like int or string
customer = this is a variable
new = this is the most important key that tells the CLR to allocate memory for a new object in managed heap. without new there is no object.
Customer() = it is constructor, a special method that runs automatically when a object is created.
        
        Heap
            +----------------------+
            | Customer Object      |
            |----------------------|
            | Name = null          |
            | Email = null         |
            +----------------------+
        since string is a reference type the default value for reference type is null.

            Stack

            +----------------------+
            | customer ----------+ |
            +--------------------|-+
                                 |
                                 ▼
                                Heap

                                +----------------------+
                                | Customer Object      |
                                |----------------------|
                                | Name = null          |
                                | Email = null         |
                                +----------------------+

        
        The variable doesnot contain the object. it contains a reference to a object.

```
