Suppose :
    Teacher says write name on your water bottle.
    The name identifies the bottle to whom it belongs to.
    The name is not bottle.
Likewise:
    Variable name idenfies a location in memory.
    The variable name is not the memory.
    The variable name is not the stored value.

Storage Location:
    When a variable is created, the computer reserves memory.
    example:
        int age;        memory is reserved even before a value is assigned.

Breaking It Down:
    int age = 25;
    age is variable name.
    memory is reserved.
    The variable age stores the value 25.

Variable = Value is Incorrect.
Variable is name and value is the data.
