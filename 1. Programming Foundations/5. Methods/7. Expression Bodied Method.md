# What Is an Expression-bodied Method?
It is a simply another way to write a method whose body consists of a single expression.

```csharp
    int Square(int number)
    {
        return number * number;
    }

    instead of this we can write
    
    int Square(int number) => number * number;

    Both are exactly same thing.

    void ShowWelcome()
    {
        Console.WriteLine("Welcome");
    }

    void ShowWelcome() => Console.WriteLine("Welcome");

    => is and lambda operator not return.
```
