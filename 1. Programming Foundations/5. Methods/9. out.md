### `bool` + `out` is most common pattern. `bool` is not compulsory

```csharp
    static void CalculateRectangle(int width, int height, out int area, out int perimeter)
    {
        area = width * height;
        perimeter = 2 * (width + height);
    }

    //usage:
    int area;
    int perimeter;
    CalculateRectangle(10,20, out area, out perimeter);
    Console.WriteLine(area);
    Console.WriteLine(perimeter)
```


## Lessons:
    - If a method need to provide one perice of information, `out` is usually not needed. it a method need to provide
    multiple pieces of information then `out` is one way to do it.
